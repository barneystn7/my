## تایید و دریافت تغییرات
برای اینکه مطمئن شوید آخرین اصلاحات اعمال شده‌اند و بتوانید آن‌ها را بررسی کنید، مراحل زیر را انجام دهید:

1. **به‌روزرسانی مخزن**
   ```bash
   git pull
   ```
   این دستور آخرین تغییرات شاخهٔ جاری را دریافت می‌کند.

2. **اجرا برای بررسی ظاهری**
   ```bash
   python VideoTool.py
   ```
   با اجرای برنامه می‌توانید ترازبندی برچسب‌ها و نحوهٔ نمایش «MB» را بررسی کنید.

3. **تبدیل به فایل اجرایی (در صورت نیاز)**
   ```bash
   python -m auto_py_to_exe
   ```
   پس از انتخاب اسکریپت، خروجی exe را بسازید و نتیجه را ببینید.

4. **گزارش نتیجه**
   اگر همه‌چیز درست بود، نیاز به اقدام دیگری نیست؛ در غیر این صورت با ذکر مشکل دوباره اطلاع دهید تا اصلاح شود.
